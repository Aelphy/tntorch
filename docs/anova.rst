anova
=====

This module supports the `analysis of variance (ANOVA) decomposition <http://www.broda.co.uk/gsa/sobol_global_sa.pdf>`_ and related techniques, such as `Sobol's method for sensitivity analysis <https://journals.plos.org/plosone/article/file?type=supplementary&id=info:doi/10.1371/journal.pone.0095610.s003>`_. 

See also the `tutorial notebook <tutorials/anova.html>`_.
